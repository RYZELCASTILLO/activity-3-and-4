## Description
This project is an Express.js server connected to a MySQL database using mysql2.

## Features
- Express server setup
- MySQL database connection
- GET all users and products
- GET user and product by ID
- Parameterized queries to prevent SQL injection

## Setup Instructions
1. Install dependencies:
   npm install
2. Create MySQL database:
   activity_db
3. Run the server:
   npm start

## Notes
Database connection depends on local MySQL configuration.
